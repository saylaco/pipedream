## Setup
Initialize project
```bash
yarn
```

Build components
```bash
yarn build
```

## Publishing Components

### Moxie
```bash
pd publish components/moxie/dist/actions/create-client/create-client.mjs
pd publish components/moxie/dist/actions/list-clients/list-clients.mjs
```
